Les indices résumés

Localisation ou dispersion

En quelques mots

  • Les séries statistiques sous souvent difficile à appréhender dans leur globalité
  • Résumer pour appercevoire la globalité
  • Limiter la perte d'information

Les paramètres de localisation

  • La moyenne
  • La médiane
  • Le mode

Les paramètres de localisation

La moyenne


a<-c(45,58,95,35,75,85,96)
sum(a)/length(a)
mean(a)

Les paramètres de localisation

La moyenne pondéré


a<-c(45,58,95,35,75,85,96)
b<-c(1,2,4,2,3,8,7)
c<-cbind(a,b)
sum(a*b)/sum(b)

Les paramètres de localisation

La médiane


a<-c(45,58,95,35,75,85,96)
median(a)
b<-c(1,2,4,2,3,8,7)
median(b)

La médiane, coupe la population en deux par égale sur une varable. Les quantilles fonctionne de la même manière

Les paramètres de localisation

Le mode


a<-c(45,48,32,35,35,58,95,35,75,85,96)
sort ( table (a), decreasing =TRUE)

le mode est la valeur de la variable qui revient le plus souvant

Les paramètres de dispersion

  • L'étendu
  • L'écart interquartile
  • La variance

Les paramètres de dispersion

L'étendu

C'est l'ensembles des valeurs que peut prendre la variable


a<-c(45,48,32,35,35,58,95,35,75,85,96)
max(a)-min(a)

Les paramètres de dispersion

L'écart interquartile

C'est l'ensembles des valeurs que peut prendre la variable


a<-c(45,48,32,35,35,58,95,35,75,85,96)
summary(a)
80-35

Les paramètres de dispersion

La variance

Elle peut être vue comme l'écart moyen à la moyenne


a<-c(45,48,32,35,35,58,95,35,75,85,96)
sum((a-mean(a))^2)/length(a)
#ce qui revient à 
mean(a^2)-mean(a)^2
#la variance dans R sans biais
var(a)
#les résultats diffères... (cf pdf en bonus dans le cour)

Les paramètres de dispersion

L'écart type

c'est une normalisation dans les unités de la série statistique de la variance


a<-c(45,48,32,35,35,58,95,35,75,85,96)
var(a)
sqrt((var(a))#racine carre de la variance
sd(a) #sd pour standard deviation

Le Q-Q plot


par(mfrow=c(2,3)) ##pour afficher deux graphiques sur la même image
Sim<-rnorm(10000, mean=0, sd=2)
boxplot(Sim)
qqnorm(Sim)
qqline(Sim)

posSkew<-rchisq(10000,6)##distribution asymmétrqiue positive
boxplot(posSkew)
qqnorm(posSkew)
qqline(posSkew)
